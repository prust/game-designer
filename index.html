Game Designer is an open-source windows application built on the LÖVE framework designed to teach programming basics via visual coding blocks (similar to Scratch! and Hour of Code). It loads levels and worlds built with the Tiled map editor and allows the user to make them interactive via dragging-and-dropping code blocks.

You can view the source on github. It is a work in progress and currently only supports the Liberated Pixel Cup tilesets found here: https://prust.github.io/tilesets/. If you have questions, you can post them on github or you can email the author, Peter Rust, at peter@peterandjessica.com.
